Lumiere — это единый клиент ИИ-чата, который объединяет все ваши любимые ИИ-модели в одном приложении. Подключайтесь к Claude, ChatGPT, Gemini, Ollama и другим — всё из единого, красиво оформленного интерфейса. Благодаря поддержке протокола OpenClaw вы получаете продвинутую оркестрацию ИИ, включая навыки, миссии и автоматизацию по расписанию.

ПОДКЛЮЧАЙТЕСЬ К ЛЮБОМУ ИИ
- Общайтесь с Claude, ChatGPT, Google Gemini, OpenRouter, Ollama и другими провайдерами
- Используйте ИИ на устройстве с Apple Intelligence (iOS 18+)
- Мгновенно переключайтесь между моделями и серверами
- Настраивайте нескольких агентов с индивидуальными идентичностями, моделями и рабочими пространствами
- Управляйте несколькими ИИ-бэкендами с простой настройкой серверов

МОЩНЫЙ ЧАТ
- Ответы в реальном времени со стримингом и богатым рендерингом Markdown
- Голосовой ввод с нативным распознаванием речи
- Вложение изображений и файлов для мультимодальных разговоров
- Встроенные инструменты: веб-поиск, загрузка веб-страниц и выполнение кода
- Более 38 слэш-команд для продвинутых пользователей
- Сохраняйте важные ответы в избранное
- Поддержка нескольких сессий с пользовательскими псевдонимами

ПРОТОКОЛ OPENCLAW
- Маркетплейс навыков — просматривайте, устанавливайте и управляйте ИИ-навыками из ClawHub
- Обучайте ИИ пользовательским навыкам с помощью кода и естественного языка
- Центр миссий — планируйте и выполняйте сложные многоэтапные задачи с ИИ-декомпозицией подзадач
- Суб-агенты — запускайте параллельных работников для распределённого выполнения задач
- Интерактивный холст — богатые HTML/JavaScript UI-компоненты на основе ИИ
- Cron-планировщик — создавайте, управляйте и отслеживайте повторяющиеся автоматизированные задачи
- Мониторинг шлюза — логи в реальном времени, состояние здоровья и конфигурация сервера
- Интеграция MCP-серверов — расширяйте возможности ИИ с помощью инструментов Model Context Protocol

АВТОМАТИЗИРУЙТЕ РАБОЧИЙ ПРОЦЕСС
- Создавайте триггеры с дип-ссылками для промптов в одно касание
- Быстрые действия на главном экране для мгновенного доступа
- Интеграция с Apple Shortcuts для бесшовной автоматизации
- Планируйте повторяющиеся задачи с помощью встроенного cron-планировщика

ПЕРСОНАЛИЗИРУЙТЕ СВОЙ ОПЫТ
- 8 потрясающих цветовых тем: Стандартная, Розовая, Зелёная, Красная, Синяя, Фиолетовая, Оранжевая и Стеклянная
- Светлый и тёмный режим с автоматическим определением системы
- Доступно на 11 языках

БЕЗОПАСНОСТЬ ПРЕЖДЕ ВСЕГО
- Биометрическая аутентификация Face ID и Touch ID
- Зашифрованное хранение API-ключей с использованием iOS Keychain
- Криптографическая идентификация устройства с аутентификацией на основе nonce
- Варианты ИИ на устройстве для полной конфиденциальности
- Никакой телеметрии или удалённого сбора данных

СОЗДАНО ДЛЯ IPAD
- Адаптивные макеты, оптимизированные для больших экранов
- Модальные окна в виде форм для естественной многозадачности
- Полная поддержка альбомной и портретной ориентации

Будь вы разработчиком, оркестрирующим ИИ-агентов, опытным пользователем, автоматизирующим сложные рабочие процессы, или кем-то, кто хочет единый доступ к лучшим ИИ-моделям — Lumiere помещает всё это в ваш карман.
