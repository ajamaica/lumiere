Lumiere — это единый клиент ИИ-чата, который объединяет все ваши любимые ИИ-модели в одном приложении. Подключайтесь к Claude, ChatGPT, Gemini, Ollama и другим — всё из единого, красиво оформленного интерфейса.

ПОДКЛЮЧАЙТЕСЬ К ЛЮБОМУ ИИ
- Общайтесь с Claude, ChatGPT, Google Gemini, OpenRouter, Ollama и другими провайдерами
- Используйте ИИ на устройстве с Apple Intelligence (iOS 18+)
- Мгновенно переключайтесь между моделями и серверами
- Управляйте несколькими ИИ-бэкендами с простой настройкой серверов

МОЩНЫЙ ЧАТ
- Ответы в реальном времени со стримингом и богатым рендерингом Markdown
- Голосовой ввод с нативным распознаванием речи
- Вложение изображений для мультимодальных разговоров
- Более 38 слэш-команд для продвинутых пользователей
- Сохраняйте важные ответы в избранное
- Поддержка нескольких сессий с пользовательскими псевдонимами

АВТОМАТИЗИРУЙТЕ РАБОЧИЙ ПРОЦЕСС
- Создавайте триггеры с дип-ссылками для промптов в одно касание
- Быстрые действия на главном экране для мгновенного доступа
- Интеграция с Apple Shortcuts для бесшовной автоматизации
- Планируйте повторяющиеся задачи с помощью cron-задач

ПЕРСОНАЛИЗИРУЙТЕ СВОЙ ОПЫТ
- 8 потрясающих цветовых тем: Стандартная, Розовая, Зелёная, Красная, Синяя, Фиолетовая, Оранжевая и Стеклянная
- Светлый и тёмный режим с автоматическим определением системы
- Доступно на 11 языках

БЕЗОПАСНОСТЬ ПРЕЖДЕ ВСЕГО
- Биометрическая аутентификация Face ID и Touch ID
- Зашифрованное хранение API-ключей с использованием iOS Keychain
- Варианты ИИ на устройстве для полной конфиденциальности
- Никакой телеметрии или удалённого сбора данных

СОЗДАНО ДЛЯ IPAD
- Адаптивные макеты, оптимизированные для больших экранов
- Модальные окна в виде форм для естественной многозадачности
