1.2.1의 새로운 기능:

• iOS 홈 화면 위젯과 Siri 제안으로 빠른 접근
• 인터랙티브 캔버스 — eval, 탐색, 프레젠트, 스냅샷 지원
• 이모지 아바타, 모델 선택기, 자동완성으로 에이전트 생성
• AI 응답을 중간에 중단하는 정지 버튼
• 세션별 채팅 입력 초안 자동 저장
• 채팅 글꼴 크기 및 글꼴 패밀리 사용자 지정
• 다른 날의 메시지 사이에 날짜 구분선
• OpenClaw 서버용 클라이언트 측 MCP 브리지
• 코드 블록의 구문 강조 및 줄 번호
• 모든 화면에서 태블릿 및 폴더블 기기 지원
• 설치됨 섹션과 활성화/비활성화 토글이 있는 스킬 마켓플레이스
• 주요 인터랙션에 햅틱 피드백 및 OTA 업데이트
• 카메라, 사진, 파일 옵션이 포함된 플러스 메뉴
• HTTP 재시도/타임아웃, 캐시 TTL, 메시지 재시도, 데이터 내보내기
• 세련된 온보딩 비주얼로 설정 화면 재디자인
• 페어링 흐름 안정성 및 기기 ID 동기화 개선
• 음성 입력, 카메라 권한, iPad 다시 렌더링 문제 수정
• 중첩 목록 렌더링, 도구 이벤트, 스킬 카드 표시 수정
• 종속성 업데이트 및 성능 개선
