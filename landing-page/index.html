Packed with Features

Everything you need to interact with AI agents, built into a native mobile experience.

Real-time Streaming Chat

WebSocket-powered conversations with AI agents. See responses as they're generated with smooth, character-by-character streaming.

Voice Input

Native speech-to-text transcription on iOS. Talk to your AI agents hands-free with real-time voice preview.

9 AI Providers

Connect to Claude, OpenAI, Ollama, OpenRouter, Apple Intelligence, Gemini Nano, Molt, and more. Switch between them seamlessly.

38 Slash Commands

Powerful command system with fuzzy autocomplete. Control your AI agents with intuitive, discoverable commands.

File Attachments

Send images, videos, and documents to your AI agents. Pick from your gallery or browse files on your device.

Rich Markdown

Full markdown rendering with syntax-highlighted code blocks, tables, blockquotes, links, and more.

Biometric Security

Protect your conversations with Face ID or Touch ID. API keys stored in secure encrypted storage.

Deep Link Triggers

Create automation triggers that launch specific AI workflows from URLs, shortcuts, or 3D Touch quick actions.

Task Scheduling

Schedule recurring agent tasks with cron-style scheduling. Automate your AI workflows to run on a timer.
