Lumiere는 여러분이 좋아하는 모든 AI 모델을 하나의 앱에 모아주는 통합 AI 채팅 클라이언트입니다. Claude, ChatGPT, Gemini, Ollama 등에 연결하세요 — 아름답게 디자인된 하나의 인터페이스에서. OpenClaw 프로토콜을 지원하여 스킬, 미션, 예약 자동화 등 고급 AI 오케스트레이션을 활용할 수 있습니다.

모든 AI에 연결
- Claude, ChatGPT, Google Gemini, OpenRouter, Ollama 및 기타 제공자와 채팅
- Apple Intelligence (iOS 18+)로 온디바이스 AI 사용
- 모델과 서버 간 즉시 전환
- 커스텀 정체성, 모델, 워크스페이스로 여러 에이전트 구성
- 간편한 서버 구성으로 여러 AI 백엔드 관리

강력한 채팅 경험
- 풍부한 마크다운 렌더링을 통한 실시간 스트리밍 응답
- 네이티브 음성 인식을 통한 음성 입력
- 멀티모달 대화를 위한 이미지 및 파일 첨부
- 내장 도구: 웹 검색, 웹 가져오기, 코드 실행
- 파워 유저를 위한 38개 이상의 슬래시 명령어
- 중요한 응답을 즐겨찾기에 저장
- 커스텀 별칭을 사용한 멀티 세션 지원

OpenClaw 프로토콜
- 스킬 마켓플레이스 — ClawHub에서 AI 스킬 탐색, 설치 및 관리
- 코드와 자연어로 AI에게 커스텀 스킬 교육
- 미션 컨트롤 — AI 기반 하위 작업 분해로 복잡한 다단계 태스크 계획 및 실행
- 서브 에이전트 — 분산 작업 실행을 위한 병렬 워커 생성
- 인터랙티브 캔버스 — AI가 구동하는 풍부한 HTML/JavaScript UI 컴포넌트
- 크론 스케줄러 — 반복 자동화 작업 생성, 관리 및 모니터링
- 게이트웨이 모니터링 — 실시간 로그, 상태 확인, 서버 구성
- MCP 서버 통합 — Model Context Protocol 도구로 AI 기능 확장

워크플로 자동화
- 원탭 프롬프트를 위한 딥링크 트리거 생성
- 빠른 접근을 위한 홈 화면 퀵 액션
- 원활한 자동화를 위한 Apple 단축어 통합
- 내장 크론 스케줄러로 반복 태스크 예약

나만의 경험 맞춤설정
- 8가지 멋진 컬러 테마: 기본, 핑크, 그린, 레드, 블루, 퍼플, 오렌지, 글래스
- 자동 시스템 감지를 통한 라이트 및 다크 모드
- 11개 언어로 제공

보안 최우선
- Face ID 및 Touch ID 생체 인증
- iOS 키체인을 통한 암호화된 API 키 저장
- 논스 기반 인증을 통한 암호화 디바이스 신원 확인
- 완벽한 프라이버시를 위한 온디바이스 AI 옵션
- 텔레메트리 및 원격 데이터 수집 없음

iPad에 최적화
- 큰 화면에 최적화된 반응형 레이아웃
- 자연스러운 멀티태스킹을 위한 폼 시트 모달
- 가로 및 세로 모드 완벽 지원

AI 에이전트를 오케스트레이션하는 개발자든, 복잡한 워크플로를 자동화하는 파워 유저든, 최고의 AI 모델에 통합 접근을 원하는 누구든 — Lumiere가 모든 것을 여러분의 주머니에 담아드립니다.
