1.2.0의 새로운 기능:

• iOS 홈 화면 위젯 — 홈 화면에서 Lumiere에 빠르게 접근
• Siri 제안 지원 — Siri 제안에 Lumiere가 표시됩니다
• 코드 블록의 구문 강조 및 줄 번호로 가독성 향상
• 태블릿 및 폴더블 기기 지원 — 모든 화면에서 최적화된 레이아웃
• 카메라, 사진, 파일 옵션이 포함된 플러스 메뉴로 쉽게 첨부
• HTTP 재시도/타임아웃, 캐시 TTL, 메시지 재시도, 전체 데이터 내보내기
• 세련된 온보딩 비주얼로 설정 화면 재디자인
• 세션 전환 시 iPad 다시 렌더링 문제 수정
• 채팅 메시지의 중첩 목록 렌더링 및 원시 마크다운 표시 수정
• 실시간 채팅 중 도구 이벤트가 표시되지 않는 문제 수정
• 검색 결과의 스킬 카드 렌더링 수정
• 종속성 업데이트 및 성능 개선
