# Simple Math Skill

Performs basic arithmetic operations safely.

## Features

- Addition, subtraction, multiplication, division
- No eval() or exec()
- Input validation
- Safe implementation

## Usage

When user requests calculations, use standard Python operators.

## Example

User: "What's 25 + 17?"
Agent: _uses simple-math skill_ The result is 42.
